# Homewrok
1.该项目能实现文本计数统计功能，能正确统计导入的纯英文txt文本中的字符数，单词数，句子数。 只要在该Cpp同文件下创建一个名称为 YLSD 的txt文件，运行后，可自动读取txt文件中的行数，字符数，句子数，单词数，并生成一个out的txt文件来存储YLSD文件中的内容以及行数，字符数，句子数，单词数的数据。

2.主要文件有 YLSD.txt 用于导入，里面的内容用于读取
  out.txt用于存放输出的内容。
